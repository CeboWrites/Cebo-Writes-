# Contributing to Cebo Writes

Thank you for your interest in contributing to Cebo Writes! This is a creative portfolio showcasing poetry, fashion, vlogs, podcasts, and newsletters.

## How to Contribute

### Content Submissions
- **Poetry**: Submit original pieces in `content/poetry/`
- **Fashion**: Add lookbooks and style essays in `content/fashion/`
- **Vlogs**: Link to video content in `content/vlogs/`
- **Podcasts**: Submit podcast information in `content/podcasts/`
- **Newsletter**: Archive editions in `content/newsletter/`

### Guidelines
1. Fork the repository
2. Create a feature branch (`git checkout -b feature/your-content`)
3. Add your content following the directory structure
4. Commit with descriptive messages
5. Push to your branch
6. Submit a Pull Request

## Code of Conduct
- Respect all contributors
- Provide constructive feedback
- Keep discussions professional and supportive
